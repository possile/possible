## Required Files:
- `entesco-v1.png` or `entesco-v1.jpg` - Screenshot of Entesco Version 1
- `entesco-v2.png` or `entesco-v2.jpg` - Screenshot of Entesco Version 2

## How to Take Screenshots:

### Windows:
1. Press `Win + Shift + S` (Snipping Tool)
2. Select area and save

### Mac:
1. Press `Cmd + Shift + 4`
2. Press `Spacebar` and click browser window

## Optimize Before Uploading:
Visit: https://tinypng.com
- Drag your images
- Download compressed versions
- Can reduce file size by 70%!

## Recommended Specs:
- **Dimensions:** 1200 x 800 pixels
- **Format:** PNG (better quality) or JPG
- **File Size:** Under 500KB each

## After Adding Images:
Open `index.html` and uncomment the image lines:
```html
<!-- Change this: -->
<div class="project-image-placeholder">01</div>

<!-- To this: -->
<img src="images/entesco-v1.png" alt="Entesco Website Version 1 Screenshot">
```
